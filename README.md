## ✨ Overview

**PaletteAI** is a next-generation **AI Design Operating System** that transforms poetic descriptions into functional design identities. More than just a color generator, it is a futuristic UI laboratory that synthesizes complete design systems, branding DNA, and immersive motion environments using **Gemini 1.5 Pro** and **Llama 3.3 70B**.

## 🚀 Core Features

- **🎭 AI Style Modes**: Instantly morph your UI between 10 visual identities: Cyberpunk, Glassmorphism, Neo Tokyo, Minimal Apple, Retro Wave, Luxury Gold, Gaming RGB, Anime Neon, AI Matrix, and Modern SaaS.
- **🧬 Brand DNA Engine**: Generate a complete visual identity and emotional persona (palette, strategy, and strategic rationale) based on brand name and industry.
- **📱 Device-Aware Live Preview**: High-fidelity virtual device frames (Mobile/Tablet/Desktop) with dynamic scaling and portrait/landscape orientation toggling.
- **🛡️ Accessibility & Contrast Engine**: Built-in WCAG 2.1 compliance validator with real-time scoring and **AI Auto-Fix** for one-click contrast correction.
- **🧪 Gradient Studio**: Create complex, high-performance mesh gradients and linear transitions with AI-optimized chromatic balance.
- **🖼️ Image DNA**: Extract chromatic signatures and emotional mood scores from visual assets to seed your design system.
- **📐 Design System Generator**: Automatically expands palettes into full design tokens: typography scales, spacing systems, border radii, and semantic component tokens.
- **✨ Interactive Motion Engine**: Cinematic motion system featuring **Magnetic Components**, **CursorGlow** lighting, and fluid morphing transitions.
- **👁️ Vision Simulation Filters**: Test designs against Protanopia, Deuteranopia, Tritanopia, and Achromatopsia using real-time SVG filters.
- **Advanced Export Engine**: High-fidelity code generation for **Tailwind Config**, **CSS Variables**, **Flutter ThemeData**, and **JSON Design Tokens**.

## 🛠️ Technology Stack

- **Frontend**: React (TSX) + Vite
- **Styling**: Tailwind CSS v4 + Global Design Token Injection
- **Animations**: `motion/react` (Framer Motion) + Liquid Morphing Engine
- **AI**: Groq Llama 3.3 70B (High-Speed Logic) & Gemini 1.5 Pro (Vision & Fallback)
- **Icons**: Lucide React
- **State Management**: Zustand (History, UI State, & Style Modes)

## 🎨 Design DNA

- **Design OS Feel**: Immersive, dark luxury UI with floating translucent panels and layered depth.
- **Typography**: Optimized for **Inter** and **Space Grotesk** with AI-driven hierarchy.
- **Kinetic Feedback**: Magnetic interactions, cursor-following lighting, and smooth spectral interpolation.

## ⚙️ Running Locally

### Prerequisites
- Node.js (Latest LTS)
- Gemini API Key ([Get it here](https://aistudio.google.com/app/apikey))
- Groq API Key ([Get it here](https://console.groq.com/))

### Setup
1. **Clone & Install**:
   ```bash
   npm install
   ```
2. **Environment Variables**:
   Create a `.env` file in the root and add your API keys:
   ```env
   VITE_GEMINI_API_KEY_1=your_key_here
   VITE_GROQ_API_KEY=your_key_here
   ```
3. **Launch**:
   ```bash
   npm run dev
   ```
